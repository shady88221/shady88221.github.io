Cred Rebranding Project Part2 Website and App

In short, it's a follow up brief from the CRED rebranding project. They are very satisfied from the previous rebranding project, so they decided to let us have a go with the phone application and website. The brief is that we need to design a financial phone application and a website with same features. The challenge in this project is the “C-LOC” system, which is a brand-new financial system that based on cryptocurrency. We need to team up with the developers and financial specialists. What’s more, the design consistency, it is the hard one we have to both consider the function and the visual element and try to make sense of complex org charts, business flows and data and ability to present information in an easy to understand and memorable way.
Therefore, firstly we discussed with the financial specialists and project manager, with their help we started our wireframe. With loads of bloody struggles we finished wirefame and then we decided to design a whole new set of different icons and image to visualize all the data and information without lose any design consistency.
